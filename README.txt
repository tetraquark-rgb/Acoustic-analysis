A program which can perform three task on a wav file (exemple file provided) :
-Analysis of the frequency of an audio file (such as the note on an instrument) and drawing the wave of the signal
-Analysis of the tristimulus which put the harmonics in three components : fundamental, harmonics 2 to 4 and harmonics 5+
-Analysis of properties with explications and exemples of what these properties mean such as the centroid (mean frequency) etc.
